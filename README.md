# UI Animation: Vanilla JS to GSAP

頻出の UI アニメーションを「Vanilla JS」から「GSAP」へ書き換えました。

## 概要

標準的な JavaScript と Web API アニメーションで実装された UI パーツを、GSAP (GreenSock Animation Platform) を用いて再構築しました。
